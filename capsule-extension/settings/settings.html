Quick-fill Presets

Click a provider to pre-fill the URL and model. Paste your own API key.

  1. Go to build.nvidia.com and sign up free — no credit card needed.
  2. Browse any model (e.g. Llama 3.1 Nemotron) → click Get API Key → copy the nvapi-… key.
  3. Click NVIDIA Build above to pre-fill the URL and model.
  4. Paste your key into the API Key field.
  5. Click Test Connection — if green, click Save Settings.
  6. Done. 40 free requests/minute, no expiry, no credit card.
