# 🫧 Capsule Hub — Chrome Extension

**Save AI conversations as smart capsules. Inject context anywhere.**

Captures conversations from ChatGPT and Claude, compresses them with any AI API, and saves two capsule types you can inject into any chat with one click — eliminating wasted tokens when a session hits the context limit.

---

## Dual Capsule System

| Type | Badge | Contents | Token Cost |
|------|-------|----------|------------|
| Raw  | 🔴 Red  | Full conversation verbatim | Up to 80,000+ tokens |
| Smart | 🔵 Blue | AI-compressed 11-point brief (~600 words) | ~500–1,000 tokens |

Both are created with a single Save click and stored locally in your browser — no server, no accounts, no data leaving your device (except the summarisation API call).

## Quick Start

### Step 1 — Generate Icons (one time only)

Open a terminal in the `capsule-extension/` folder and run:

```bash
python generate-icons.py
```

This creates `icons/icon16.png`, `icons/icon48.png`, `icons/icon128.png` using only Python's standard library — no pip install needed.

### Step 2 — Load the Extension in Chrome

1. Open Chrome and go to `chrome://extensions`
2. Enable **Developer Mode** (toggle, top-right corner)
3. Click **Load unpacked**
4. Select the `capsule-extension/` folder
5. Pin the extension from the puzzle piece 🧩 icon in the toolbar

### Step 3 — Configure Your API Key

1. Click the Capsule Hub icon → **⚙️ Settings**
2. Click a provider preset to auto-fill URL + model
3. Paste your API key
4. Click **Test Connection** → green ✅ → **Save Settings**

> **Recommended free option — NVIDIA Build:**
> - Sign up at [build.nvidia.com](https://build.nvidia.com) (no credit card)
> - Browse any model → **Get API Key** → copy `nvapi-…`
> - Preset: `https://integrate.api.nvidia.com/v1` · Model: `nvidia/llama-3.1-nemotron-ultra-253b-v1`
> - 40 free requests/minute, no expiry

### Step 4 — Save Your First Capsule

1. Visit [chatgpt.com](https://chatgpt.com) or [claude.ai](https://claude.ai) and have a conversation
2. Click the Capsule Hub extension icon
3. (Optional) Type a name, or leave blank for auto-naming
4. Click **💾 Save** — both capsules are created

### Step 5 — Inject a Capsule

1. Open a fresh ChatGPT or Claude tab (new conversation)
2. Click the extension icon
3. Find your capsule in the list
4. Click **🔴 Raw** (full text) or **🔵 Smart** (compressed brief)
5. The text is injected into the chat input — ready to send

## Privacy & Security

- **No server** — zero data transmission except the AI summarisation call
- **API key** stored only in `chrome.storage.local`, never in code or URLs
- **No analytics, no telemetry, no remote logging**
- Content scripts scoped only to `chatgpt.com` and `claude.ai`
- No `eval()`, no remote script loading
- Minimal permissions: `storage`, `activeTab`, `scripting`
